## CSS 的三大特性

### 层叠性

层叠性: 对于相同选择器中的相同属性，哪个选择器离标签近 就使用哪个属性，称为就近原则 层叠也叫覆盖
`层叠性作用是为了解决样式冲突问题`

### 继承性

继承性: 子承父业
作用: 简化代码
可以继承的属性: font- text- line- color

### 优先级

- 不同的选择器作用于同一个元素上，权重最高的那个被优先使用

- 继承: 0 标签: 1 类名: 10 ID: 100 行内:1000 !important +∞

```text
div ul li --->  0 0 0 3
.nav ul li ---> 0 0 1 2
a:hover ---> 0011
.nav a  --- 0011
```
